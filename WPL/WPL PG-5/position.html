CSS RELATIVE POSITIONING

An element with position: relative; is offset from its original position. Critically, it does not affect the layout of other surrounding elements, preserving their space in the normal document flow.

CSS ABSOLUTE POSITIONING

An element with position: absolute; is removed from the normal document flow and positioned relative to the nearest ancestor that has a position property other than static. Here, the positioning ancestor is the div with class "absolute-container", which is set to relative.
